CODE NET是为代码分析所建立的源代码语料库。在软件工程领域中，源代码是一个主要的研究对象，但是长期以来缺少标准的代码预料库。通过分享我们认为可能是世界上最好的代码，我们希望能在代码分析相关领域建立一个开放的标准。我们的数据来源于github和stackoverflow上的代码片段，这些代码片段集中了编程人员在多年实践中的智慧结晶。
